# 🐯 한경 워드퍼즐 (HKNU Word Puzzle)

한경국립대학교 관련 낱말퍼즐 게임

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)
![Vite](https://img.shields.io/badge/Vite-6-646CFF?logo=vite)
![SASS](https://img.shields.io/badge/SASS-CC6699?logo=sass)

## 🎮 게임 소개

한경국립대학교를 얼마나 알고 있나요? 힌트를 보고 한경국립대 관련 단어를 맞춰보세요!

### 주요 기능
- 📝 **2개 스테이지 낱말퍼즐** - Stage 1 (기초), Stage 2 (심화)
- ⏱️ **실시간 타이머** - 클리어 시간 측정
- 🎯 **정답/오답 피드백** - 색상 변화 + 애니메이션
- ❓ **추가 힌트 기능** - 1회 이상 틀린 단어에 정답 절반 힌트 제공
- 🏆 **온라인 랭킹 시스템** - Supabase 기반 실시간 랭킹
- 📱 **반응형 디자인** - 모바일, 태블릿, 데스크톱 지원
- ♿ **웹 접근성** - 키보드 네비게이션, ARIA 레이블

## 🎯 스테이지 정보

### Stage 1: 한경국립대 기초 (🟢 Easy)
| 번호 | 단어 | 방향 | 힌트 |
|------|------|------|------|
| 1 | 백호 | → | 한경국립대학교의 상징 동물, 흰 호랑이 |
| 2 | 안성 | → | 한경국립대 안성캠퍼스 소재 도시 |
| 3 | 포도 | → | 안성 5대 특산물 중 하나, 보라색 과일 |
| 4 | 국립대 | → | 국가가 설립한 대학교 유형 |
| 5 | 은행 | ↓ | 한경국립대학교의 교목(校木), 노란 잎이 특징인 나무 |

### Stage 2: 한경국립대 & 안성 심화 (🔴 Hard)
| 번호 | 단어 | 방향 | 힌트 |
|------|------|------|------|
| 1 | 한경이 | → | 백호를 모티브로 한 한경국립대 공식 마스코트 |
| 2 | 평택 | → | 한경국립대 제2캠퍼스 소재 도시 |
| 3 | 바우덕이 | → | 안성 출신, 조선 후기 남사당패의 여성 꼭두쇠 |
| 4 | 안성맞춤 | → | "딱 맞다"는 의미, 안성 유기(鍮器)에서 유래 |
| 5 | 남사당 | ↓ | 조선시대 유랑 예인 집단, 안성이 발상지 |

## 🧩 핵심 컴포넌트 설명

### `usePuzzle` 훅
```javascript
const {
  grid,              // 2D 그리드 배열
  hints,             // { across: [], down: [] }
  completedCells,    // Set - 완료된 셀
  completedWords,    // Set - 완료된 단어
  activeCell,        // { row, col } - 현재 포커스 셀
  selectedWordId,    // 선택된 단어 ID
  progress,          // 진행률 (0-100)
  isPuzzleComplete,  // 퍼즐 완료 여부
  handleCellInput,   // 셀 입력 처리
  handleCellFocus,   // 셀 포커스 변경
  handleHintClick,   // 힌트 클릭 처리
  moveToNextCell,    // 다음 셀로 이동
} = usePuzzle(stageNumber);
```

### `GameContext`
```javascript
const {
  currentStage,    // 현재 스테이지 번호
  gameStatus,      // 'idle' | 'playing' | 'paused' | 'cleared'
  time,            // 경과 시간 (ms)
  isRunning,       // 타이머 실행 중 여부
  clearTime,       // 클리어 시간
  isNewRecord,     // 신기록 여부
  startGame,       // 게임 시작
  pauseGame,       // 일시정지
  clearStage,      // 스테이지 클리어
  resetGame,       // 게임 리셋
} = useGame();
```
